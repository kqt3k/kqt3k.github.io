⛭ Принцип BrokenBase

  • • Магическая пара: "xuy" ↔ " ) " (точное соответствие багу Base64).
  • • Универсальный алгоритм: XOR с фиксированным ключом 0x7A (122).
  • • Обратимость: decode(encode(текст)) = исходный текст (включая любой UTF-8).
  • • Эффект кракозябр: результат состоит из спецсимволов, нечитаемых человеком.
  • • Полностью локально: данные никогда не покидают ваш браузер.

⤚ Примеры трансформации

encode "xuy" → " ) "  |  decode " ) " → "xuy"
encode "Hello" → "I~I" (пример кракозябр)
decode кракозябр → оригинал "Hello"
💡 Для любых символов (русские, эмодзи) работает идентично.
